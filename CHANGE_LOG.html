Change Log#

Change log from V2021 to V2022

Constraint#

  • Coupling and ConstraintModel.Coupling(): add alpha argument.

  • Tie and ConstraintModel.Tie(): arguments master and slave change to main and secondary.

Datum#

  • Add DatumCsys.globalToLocal() and DatumCsys.localToGlobal().

Field Report#

  • FieldReportOptions.setValues(): add printLocalCSYS argument.

Interaction#

  • ContactExp and InteractionModel.ContactExp(): argument masterSlaveAssignments renames to mainSecondaryAssignments.

  • ContactStd and InteractionModel.ContactStd(): argument masterSlaveAssignments renames to mainSecondaryAssignments.

  • CyclicSymmetry and InteractionModel.CyclicSymmetry(): arguments master and slave rename to main and secondary.

  • ExpInitialization and InteractionModel.ExpInitialization(): argument slaveNodesetName renames to secondaryNodesetName.

  • GapHeatGeneration: argument slaveFraction renames to secondaryFraction.

  • InitializationAssignment.appendInStep(): argument assignments’s docstring changes (slave -> secondary).

  • MasterSlaveAssignment change to MainSecondaryAssignment.

  • InteractionModel.contactDetection(): arguments createUnionOfMasterSurfaces, createUnionOfSlaveSurfaces, createUnionOfMasterSlaveSurfaces change to createUnionOfMainSurfaces, createUnionOfSecondarySurfaces, createUnionOfMainSecondarySurfaces.

  • PolarityAssignments.changeValuesInStep(): argument stepName’s docstring changes (master-slave -> main-secondary).

  • PressurePenetration and InteractionModel.PressurePenetration(): arguments masterPoints and slavePoints rename to mainPoints and secondaryPoints.

  • Radiation and InteractionModel.Radiation(): arguments masterEmissivity and slaveEmissivity rename to mainEmissivity and secondaryEmissivity.

  • SelfContactStd and InteractionModel.SelfContactStd(): argument smooth’s docstring changes (master -> main).

  • StdContactControl and InteractionModel.StdContactControl(): argument uerrmx’s docstring changes (slave -> secondary).

  • SurfaceToSurfaceContactExp and InteractionModel.SurfaceToSurfaceContactExp(): arguments master and slave rename to main and secondary.

  • SurfaceToSurfaceContactExp.swapSurfaces(): docstring changes (master -> main, slave -> secondary).

  • SurfaceToSurfaceContactStd and InteractionModel.SurfaceToSurfaceContactStd(): arguments master and slave rename to main and secondary.

  • SurfaceToSurfaceContactStd.swapSurfaces(): docstring changes (master -> main, slave -> secondary).

Job#

  • JobFromInputFile and JobMdb.JobFromInputFile(): add argument licenseType.

  • JobMdb.Job(): add argument licenseType.

  • Add Coexecution to attributes of Mdb.

Material#

  • Add classes CrushStress and CrushStressVelocityFactor, add method Material.CrushStress().

  • Elastic and Material.Elastic(): argument type add BILAMINA.

  • Hyperelastic and Material.Hyperelastic(): argument type add VALANIS_LANDEL.

  • Plastic and Material.Plastic(): add argument extrapolation.

Mesh#

  • ElemType: add arguments linearKinematicCtrl and initialGapOpening.

  • MeshFaceArray: argument faces changes to elemFaces.

  • MeshEdgeArray: argument edges changes to elemEdges.

Odb#

  • Add OdbDatumCsys.globalToLocal() and OdbDatumCsys.localToGlobal().

Optimization#

  • BeadTask and OptimizationTaskModel.BeadTask(): add argument groupOperator.

  • DesignDirection and and OptimizationTask.DesignDirection(): arguments masterPoint and masterDetermination change to mainPoint and mainPointDetermination.

  • DrillControl and and OptimizationTask.DrillControl(): arguments masterPoint and masterDetermination change to mainPoint and mainPointDetermination.

  • ShapeDemoldControl and and OptimizationTask.ShapeDemoldControl(): argument masterDetermination changes to mainPointDetermination.

  • ShapeMemberSize and and OptimizationTask.ShapeMemberSize(): add arguments assignNodeGroupRegion and nodeGroupRegion.

  • ShapePointSymmetry and and OptimizationTask.ShapePointSymmetry(): argument masterDetermination changes to mainPointDetermination.

  • ShapeRotationalSymmetry and and OptimizationTask.ShapeRotationalSymmetry(): arguments masterPoint and masterDetermination change to mainPoint and mainPointDetermination.

  • ShapeTask and OptimizationTaskModel.ShapeTask(): add argument groupOperator.

  • SizingTask and OptimizationTaskModel.SizingTask(): add argument groupOperator.

  • StampControl and and OptimizationTask.StampControl(): arguments masterPoint and masterDetermination change to mainPoint and mainPointDetermination.

  • TopologyTask and OptimizationTaskModel.TopologyTask(): add argument groupOperator.

  • Add TopologyMillingControl and OptimizationTask.TopologyMillingControl().

  • TurnControl and and OptimizationTask.TurnControl(): arguments masterPoint and masterDetermination change to mainPoint and mainPointDetermination.

Part#

  • PartBase.getCoordinates() add argument csys.

Session#

  • Session.printToFile(): argument compression’s docstring changes.

Step Output#

  • FieldOutputRequest.setValuesInStep(): argument timePoints changes to timePoint.

XY#

  • XYData.xyDataListFromField() add argument operator.
